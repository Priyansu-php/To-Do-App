# 📝 TaskFlow - Modern Todo App

A clean, responsive, and modern Todo application built with **React**, **Context API**, and **Tailwind CSS**. TaskFlow helps users organize daily tasks with a smooth user experience while demonstrating efficient state management using React Context API.

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Persistent storage using Local Storage
- ⚛️ Global state management with Context API
- 📱 Responsive UI
- 🎨 Modern interface built with Tailwind CSS

---

## 🛠 Tech Stack

- React.js
- Context API
- Tailwind CSS
- JavaScript (ES6+)
- Local Storage

```bash

```



